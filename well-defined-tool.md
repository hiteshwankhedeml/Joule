# Well defined tool

1. Must be independently tested and operational
2. Define its Purpose in Plain Business Language
3. Craft Agent Instructions as a Tool Usage Guide for a New Team Member

* The core instructions you give your agent in Joule Studio are not just a high-level goal; they are its operational playbook.&#x20;
* This playbook must teach the agent how to strategically use the specific tools you have given it.
* Think of it this way: When a new employee joins, you don't just say "Help customers."&#x20;
* You show them the software they will use and explain how and when to use each application.&#x20;
* Your agent's instructions must do the same.
* Your instructions should be a guide that teaches the agent to reason about its tools:
  * **Tool Selection Strategy** (Which tool for which job?)
  * **Tool Sequencing** (In what order should I use them?)
  * **Data Source Prioritization**(Which tool do I trust the most?)
  * **Handling Tool Limitations** (What to do when a tool isn't enough?)
