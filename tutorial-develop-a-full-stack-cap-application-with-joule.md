# Tutorial: Develop a Full-Stack CAP Application with Joule

* Joule is a Generative AI assistant that will create code for you

<mark style="color:purple;background-color:purple;">**Step 1: Create a multi-step CAP application**</mark>

* Open your Full Stack Node project.
* In SAP Business Application Studio, the SAP Build Code development environment, open the digital assistant, Joule, from the activity bar.
*

    <figure><img src=".gitbook/assets/image (6).png" alt=""><figcaption></figcaption></figure>
* &#x20;select the ’/cap-gen-app\` ⇒ Enter prompt ⇒ Create a customer loyalty application with sample data and application logic.
*

    <figure><img src=".gitbook/assets/image (7).png" alt=""><figcaption></figcaption></figure>
* It will propose the steps

1. Generate CAP data model
2. Generate CAP sample data
3. Generate CAP application logic

* We can either modify prompt generated for each or can run the prompt generated for each step
