# What is SAP Joule?

* AI powered co-pilot designed to simplify work across the SAP ecosystem
* With natural language integration and consistent user experience, Joule helps you get tasks done faster - wherever you are in SAP
* Built into SAP cloud solutions like Hana, Successfactor, enables users to ask questions, retrieve insights and take actions - all in one seamless flow
* Whether checking HR requests or retrieving business data, Joule responds instantly and intelligently
