# Joule Agents - Components

**Goals & Instructions Module:** Agents purpose and using behaviour using natural language

**Planning & Reasoning Engine:** Powered by LLM, analyzes task, breaks into smaller objectives and decides action plan based on context and tools available

**Tools:** Joule skills, Calculator and other utilities

**Memory Module:** Store past experience and memories enabling the agent to improve over time

**Integration Layer:** To communicate with SAP and non SAP system for data access etc
